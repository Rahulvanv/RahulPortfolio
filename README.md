# RahulPortfolio

A dynamic Java Developer Portfolio Website built using core Java web technologies. This project showcases my skills, education, projects, experience, certifications, and contact information using JSP, Servlets, JDBC, and MySQL.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* JSP
* Java Servlet
* JDBC
* MySQL
* Apache Tomcat
* Eclipse IDE

## Features

* Responsive portfolio homepage
* About Me section
* Skills section
* Education section
* Experience section
* Certifications section
* Projects section
* Contact section
* MVC architecture using Java
* Database connectivity using JDBC
* MySQL integration

## Software Requirements

* Java JDK 8 or above
* Eclipse IDE
* Apache Tomcat 9
* MySQL Server
* MySQL Connector/J

## How to Run

1. Clone this repository.
2. Import it into Eclipse as a Dynamic Web Project.
3. Configure Apache Tomcat.
4. Create the required MySQL database.
5. Update the database credentials in `DBConnection.java`.
6. Add the MySQL Connector JAR (if needed).
7. Run the project on Tomcat.
